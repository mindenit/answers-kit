## Answers Kit

Mindenit Answers API client for Node.js

## Installation

```sh
npm install @mindenit/answers-kit
# yarn add @mindenit/answers-kit
# pnpm add @mindenit/answers-kit
```

## Usage

```js
import { AnswersKit } from '@mindenit/answers-kit';

const kit = new AnswersKit();
```
